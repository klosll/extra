Price Unit with Tax in Sale Order Lines

Display unit price with taxes included

This module adds a new field to display the unit price with taxes included in sale order lines.

Features

Automatic Calculation

The unit price with tax is automatically calculated based on the unit price, taxes, and discounts applied.

Easy to Use

The new field is displayed right after the standard unit price field in both tree and form views.
