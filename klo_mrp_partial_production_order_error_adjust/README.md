Odoo Manufacturing Resource Planning. Partial Production Order error adjust
----------------------------------------------------------------


En el código nativo de Odoo, al realizar una orden de producción parcial,
salta un error: ¡La cantidad a producir debe ser positiva!
Esto no ocurre en la versión de Odoo de Docker.
Aquí se han traido los cambio de la versión de Docker para aplicarlos a la nativa.
